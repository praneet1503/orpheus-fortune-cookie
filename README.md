# Orpheus Fortune Cookie
this is a simple aniimation project that i made to learn more about animation and how to use it in a project. i used the gsap library to create the animation and i used html and css to create the structure and style of the project. i also created pixelated sprites for the cookie and the orpheus character. the animation is a simple one where the cookie breakes and the orpheus character comes out of it. i also added some sound effects to make it more fun. i hope you enjoy it!
## How to use
```zsh
 git clone https://github.com/praneet1503/orpheus-fortune-cookie.git
```
install the extension of live server in the vs.code if you dont have that 
[(https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)]

and then open the index.html file in the live server and you can see the animation
